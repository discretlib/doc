Sommaire

Ce document devrait être utilisé après avoir suivi un des tutoriels afin de comprendre les bases de Discret.

Il décrit les différents concepts que vous devrez connaître pour utiliser Discret efficacement. Contrairement aux tutoriels, il ne fournit pas de code directement utilisable.

  • Gestion des Données
    Dans cette section vous apprendrez à créer un modèle de données et à l'utiliser
  • Droits d'accès
    Dans cette section vous apprendrez à gérer les droits d'accès requis pour la synchronisation
  • Configuration
    Cette section décrit les options de configuration et comment installer un serveur de rencontre
